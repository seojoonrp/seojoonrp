[컴퓨터의 개념 및 실습] 2024-14433 최서준, 2024-13956 최지훈 벽돌깨기 게임 개발 프로젝트입니다.
사용 언어 : Python


[게임 방법]
맨 처음 화면에서, 아래에 있는 세 맵들 중 하나를 클릭하면, 그 맵으로 이동하여 게임을 플레이할 수 있습니다.
총 세 개의 목숨이 주어지며, 각 목숨마다 얻은 점수를 합한 것이 최종 점수가 됩니다.
각 목숨마다, 마우스를 임의 방향으로 클릭하면 공이 발사됩니다. 공은 여러 블럭에 부딪히며 블럭들을 파괴하고, 플레이어는 패들을 움직여 공이 스크린 밑으로 떨어지지 않도록 조정하면 됩니다.
패들 조정은 좌우 화살표 키로 할 수 있으며, 공이 스크린 밑으로 떨어지면 목숨을 잃습니다. 참고로, 각 목숨마다 공은 시간이 지날수록 빨라집니다.

게임에는 총 6개의 블럭이 있습니다.
1) 일반 블럭 : 1부터 3까지의 목숨을 가지는 회색 블럭으로, 목숨은 중앙의 원 개수로 확인할 수 있습니다. 특수 효과 없이 목숨이 모두 사라지면 파괴됩니다.
2) 고정 블럭 : 맵 2, 3에 등장하는 검은색 블럭으로, 아무리 맞춰도 파괴할 수 없습니다.
3) 폭발 블럭 : 10.0%의 확률로 등장하는 빨간색 블럭으로, 파괴 시 십자가 방향의 4개 블럭을 추가로 파괴합니다.
4) 통과 아이템 블럭 : 6.67%의 확률로 등장하는 파란색 블럭으로, 파괴 시 통과 아이템을 생성합니다. 통과 아이템을 획득하면 일정 시간 동안 공이 모든 블럭을 통과하며 파괴합니다. (고정 블럭 제외)
5) 패들 크기 증가 아이템 블럭 : 10.0%의 확률로 등장하는 초록색 블럭으로, 파괴 시 패들 크기 증가 아이템을 생성합니다. 패들 크기 증가 아이템을 획득하면 일정 시간 동안 패들의 크기가 커집니다.
5) 패들 크기 감소 아이템 블럭 : 10.0%의 확률로 등장하는 노란색 블럭으로, 파괴 시 패들 크기 감소 아이템을 생성합니다. 패들 크기 감소 아이템을 획득하면 일정 시간 동안 패들의 크기가 작아집니다.
각 블럭을 파괴할 때마다 10점을 얻습니다. 공이 패들에 다시 닿을 때 블럭들은 다시 랜덤으로 재생되며, 총 개수는 9개로 일정합니다. (고정 블럭 제외)

목숨을 모두 잃으면 이름을 입력할 수 있고, 순위를 확인할 수 있습니다. 이후 스코어보드 화면으로 넘어가면 각 맵마다 top 5 기록을 확인할 수 있습니다.

최대한 높은 점수를 기록해서 스코어보드에 자신의 이름을 남겨보세요. 행운을 빕니다 :)
